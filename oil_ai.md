# 이란 전쟁 유가 변동 · 한국 자취생 생활비 영향 분석

## 프로젝트 개요

이란 전쟁으로 인한 유가 변동이 한국 자취생 생활비에 미치는 영향을 
실제 데이터로 분석하고 시나리오별로 예측하는 Streamlit 웹앱

---

## 기술 스택

- **언어**: Python 3.11+
- **웹 프레임워크**: Streamlit
- **데이터 분석**: Pandas, NumPy
- **시각화**: Plotly
- **배포**: GitHub + Streamlit Cloud

---

## 디자인 방향

Bloomberg / Reuters 스타일 금융 뉴스 대시보드

- 배경: `#0A0E1A` (딥 네이비)
- 카드: `#111827` (다크 그레이)
- 강조색 1: `#EF4444` (레드) → 전쟁/상승 관련
- 강조색 2: `#3B82F6` (블루) → 일반 지표
- 강조색 3: `#10B981` (그린) → 안정/하락
- 텍스트: `#F9FAFB` (흰색)
- 보조 텍스트: `#9CA3AF` (회색)
- 폰트: 숫자는 모노스페이스 느낌
- 그래프: 다크 테마 (plotly dark template)

### 헤더 (상단 고정)
```
🔴 LIVE | 이란-이스라엘 전쟁 진행중 | 
브렌트유 $108.64 ▲43% | 휘발유 2,011원/L ▲13% | 생활물가 +4.3%
```
- 실시간 뉴스 티커 느낌
- 전쟁 관련 수치는 빨간색 강조
- 상단 고정 (sticky)

### 공통 규칙
- 각 탭 상단에 KPI 카드 4~5개 배치
- 숫자 크고 굵게 표시
- 상승은 🔴 빨간색, 하락은 🟢 초록색
- 카드에 그림자 효과
- 출처 항상 하단에 명시

## 데이터 파일

```
iran_oil_app/
├── app.py
├── CLAUDE.md
└── data/
    ├── chart_20260607T031456.csv          ← Macrotrends 브렌트유
    ├── 생활물가지수_2020100__20260607122851.xlsx  ← KOSIS 생활물가
    └── 소비자물가지수_2020100__20260607123034.xlsx ← KOSIS 소비자물가
```

### 데이터 설명

**브렌트유 CSV (Macrotrends)**
- 컬럼: Date, Value
- 날짜 형식: MM/DD/YYYY
- 2022년~2026년 데이터 사용
- 출처: macrotrends.net

**생활물가지수 Excel (KOSIS)**
- 행: 품목 (라면, 달걀, 우유, 식용유, 두부 등)
- 열: 연월 (2022.01 ~ 2026.05)
- 기준: 2020=100
- 출처: 통계청 KOSIS

**소비자물가지수 Excel (KOSIS)**
- 전체 소비자물가 월별 데이터
- 출처: 통계청 KOSIS

## 탭 구성 (5개)

### 탭 1: 📊 유가 분석

**KPI 카드 (상단)**
- 현재 브렌트유 가격
- 이란전쟁 이후 상승률
- 현재 국내 휘발유 가격
- 러-우전쟁 대비 비교

**그래프 1: 브렌트유 월별 추이 (2022~2026)**
- Plotly 라인 차트 (다크 테마)
- 러-우전쟁 구간 (2022.02~2022.06) 파란색 음영
- 이란전쟁 구간 (2026.02~현재) 빨간색 음영
- 평균선 점선으로 표시
- 출처: Macrotrends

**그래프 2: 국내 휘발유 가격 추이 (2022~2026)**
- Plotly 라인 차트
- 전쟁 구간 강조
- 출처: 한국석유공사 오피넷

---

### 탭 2: 🛒 물가 변화

**KPI 카드 (상단)**
- 라면 / 달걀 / 우유 / 식용유 / 두부
- 각각 2022.01 대비 현재 변화율 (%)

**그래프 1: 품목별 물가지수 추이**
- Plotly 멀티 라인 차트
- 품목별 다른 색상
- 전쟁 구간 음영 표시
- 기준선 (100) 점선
- 출처: 통계청 KOSIS 생활물가지수

**그래프 2: 연도별 품목 비교 바 차트**
- 2022 / 2023 / 2024 / 2025 / 2026 비교
- 출처: 통계청 KOSIS

---

### 탭 3: 🔮 시나리오 예측

**시나리오 3가지**
```
⚡ 국지전      → 유가 +25%
🚢 호르무즈 봉쇄 → 유가 +60%  
💥 전면전      → 유가 +110%
```

**시나리오별 탄성계수 (유가 1% 오를 때 품목 가격 변화율)**
```
휘발유:   0.70  (연료 직결)
택배비:   0.35  (운송비 연동)
라면:     0.15  (포장재+운송)
달걀:     0.08  (운송비만)
우유:     0.08  (운송비+포장)
식용유:   0.30  (원료 직결)
생수:     0.12  (플라스틱병)
샴푸:     0.25  (플라스틱 용기)
가스요금: 0.20  (LNG 연동)
전기요금: 0.15  (발전연료)
```

**품목별 현재 가격 (기준값)**
```
휘발유:   2,011원/L
택배비:   2,500원/건
라면:     950원/봉
달걀:     6,500원/30구
우유:     2,200원/L
식용유:   4,500원/900ml
생수:     1,200원/2L
샴푸:     8,000원/개
가스요금: 55,000원/월
전기요금: 45,000원/월
```

**화면 구성**
- 왼쪽: 시나리오 선택 버튼 3개
- 선택 시 오른쪽에 품목 카드 10개 표시
  - 현재가격 (취소선)
  - 예측가격 (큰 글씨, 강조색)
  - 상승액 뱃지
- 하단: 한달 추가 생활비 요약 카드
- 하단: 시나리오별 비교 바 차트

---

### 탭 4: 🧮 생활비 계산기

**기능**
- 소비자가 월 소비량 직접 입력
- 시나리오 선택 (드롭다운)
- 자동으로 현재 vs 예측 생활비 계산

**입력 항목 (12개)**
```
라면 (봉), 달걀 (판), 우유 (개)
식용유 (병), 생수 (병), 두부 (모)
샴푸 (개), 세제 (개), 화장지 (묶음)
버스요금 (회), 전기요금 (월), 가스요금 (월)
```

**출력**
- 현재 월 생활비 합계
- 예측 월 생활비 합계
- 추가 부담 금액 (강조)
- 연간 추가 부담
- 품목별 추가 부담 가로 바 차트

---

### 탭 5: 🤖 AI 챗봇

**기능**
- 시나리오 선택 사이드바
- 키워드 기반 답변
- 빠른 질문 버튼 5개

**답변 가능 키워드**
```
휘발유, 기름값, 라면, 달걀, 우유
식용유, 택배, 배송, 항공, 여행
난방, 가스, 생활비, 한달
호르무즈, 걸프전, 과거, 비교
플라스틱, 포장
```

**빠른 질문 버튼**
```
한달 얼마나 더 나와요?
라면값은요?
휘발유는요?
식용유 많이 올라요?
호르무즈가 뭐예요?
```
